# JavaScript Calculator

A simple and interactive calculator built using HTML, CSS, and JavaScript. This project demonstrates basic arithmetic operations and showcases my skills in frontend web development.

## Features

- Perform basic arithmetic operations: addition, subtraction, multiplication, and division.
- Responsive design, works seamlessly on both desktop and mobile devices.
- Clear and intuitive user interface.
- Real-time display of inputs and results.

## Technologies Used

- HTML: Structure of the calculator.
- CSS: Styling and layout.
- JavaScript: Core functionality and logic.
